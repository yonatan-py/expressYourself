#expressYourself

scaffold for expressjs with coffeescript and jade

Use:

    git clone https://github.com/yonatan-py/expressYourself <app-name>
    cd <app-name>
    sudo ./install.sh
    sudo chmod 776 ./start.sh
    ./start.sh

go to:

    localhost:3000

to see app.

go to: 

    localhost:5000/bracket
    
to see editor.
